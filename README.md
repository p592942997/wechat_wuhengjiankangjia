小牛Admin v1.0.3
===============

> 运行环境要求PHP7.1+。

## 主要新特性

* 项目使用thinkphp6+layui搭建
* 整合七牛云、阿里云oss
* 分组权限系统
* 图片上传组件
* 系统参数配置写入配置文件
* 封装美化常用的js方法

## 安装

~~~
手动创建数据库，导入根目录下的db.sql，修改根目录下的.env文件配置数据库信息
~~~

运行目录
~~~
public
~~~
